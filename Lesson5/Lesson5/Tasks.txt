К классу Array добавить следующий функционал:
1.1. Конструктор, который принимает размер массива n и число v. Создает массив размера n, все элементы которого 
заполнены значением v.
1.2. Конструктор, который принимает размер массива n и два числа min, max. Создает массив размера n
который заполнен случайными  числами в джиапазоне [min, max].

1.3. Добавить метод IsSorted, который возвращает true, если массив отсортирован по возрастанию, иначе false.


Написать класс DynamicArray.
Он также как и Array хранит массив, но при этом может менять свой размер во время жизни.
2.1. У него такие же конструкторы как и у Array.
2.2. Такие же методы SetValue, GetValue, Size.

2.3. Метод SetNewSize(int new_size). Изменяет размер массива. Если указан больший размер, то новые элементы заполнить нулями.

2.4. Метод PushBack(int value). Добавляет value в конец массива и увеличивает размер массива на 1.
Например был массив 1 2 3 4. Вызвали PushBack(13). Массив стал 1 2 3 4 13.

2.5. Метод PopBack(). Выкидывает из массива последний элемент. Уменьшает размер на 1.

Примечаение: В с++ нет встроенной команды изменения размера динамической памяти. 
Так что тебе придется попросить новую, скопировать туда данные и очисить (delete[]) старую.
